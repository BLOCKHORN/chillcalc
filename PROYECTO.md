# 🍏 Easypocket Terminal (v2.0)

**Easypocket** es una infraestructura financiera de nueva generación diseñada para quienes exigen control absoluto, elegancia y precisión técnica. Inspirada en los estándares de diseño de **Apple** y las fintech de élite, esta terminal transforma la gestión de capital en una experiencia de alta fidelidad.

---

## 💎 Identidad Visual: Apple-Emerald Elite
La aplicación ha sido reconstruida bajo una estética **Institutional Elite**, priorizando el minimalismo, la profundidad física y la legibilidad monumental.
- **Modo OLED:** Un fondo negro puro que resalta los datos y reduce el consumo.
- **Esmeralda Corporativo:** El hilo conductor visual que indica integridad y crecimiento.
- **Glassmorphism:** Capas de cristal esmerilado con desenfoques masivos (`backdrop-blur-3xl`).
- **Animaciones Orgánicas:** Transiciones fluidas basadas en curvas de aceleración de iOS.

---

## 🧠 Capacidades del Sistema

### 1. Overview (Dashboard Inteligente)
- **Motor de Integridad:** Visualización de patrimonio neto en tiempo real sincronizado directamente con la base de datos.
- **IA Insights:** Heurística avanzada que detecta fugas de gasto, tendencias de ahorro y anomalías mensuales.
- **Performance Global:** Gráficos de área de alta precisión para monitorizar la evolución histórica del capital.
- **Presupuestos Adaptativos:** Límites por categoría con barras de energía dinámicas que alertan sobre el consumo de capital.

### 2. Entities (Gestión de Cuentas)
- **Branding Dinámico:** Reconocimiento automático de entidades bancarias (BBVA, ING, Revolut, Santander, etc.) con carga de logotipos oficiales.
- **Integridad Atómica:** Los saldos se calculan exclusivamente mediante transacciones justificadas, eliminando errores de cálculo humanos.
- **Activos Consolidados:** Soporte para cuentas bancarias, efectivo, remuneradas y carteras de inversión.

### 3. Audit Log (Movimientos)
- **Registro de Auditoría:** Un historial técnico detallado con filtros laterales rápidos por periodo y categoría.
- **Trazabilidad de Entidades:** Cada movimiento muestra visualmente la marca de la cuenta origen para una identificación instantánea.

### 4. Shared Engine (Colaboración Pro)
- **Arquitectura Distribuida:** Motor estilo Splitwise para gestionar gastos compartidos con total transparencia.
- **Resolución de Conflictos:** Algoritmo que calcula el número mínimo de pagos necesarios para liquidar deudas.
- **Generated Access:** Generación de enlaces con tokens de seguridad únicos para invitar a socios externos a ver el balance sin registro.

### 5. Institutional Planning (Simuladores)
- **FIRE Simulator:** Proyección técnica para la independencia financiera basada en la regla del 4% y ROI neto.
- **Interest Calculator:** Herramienta avanzada que visualiza el efecto bola de nieve, separando capital aportado de intereses generados.

### 6. Timeline & Journal (Calendario)
- **Cash Flow Analysis:** Calendario mensual que proyecta entradas y salidas de capital.
- **Bitácora Financiera:** Sistema de notas persistentes en la nube vinculadas a días específicos para aportar contexto a la auditoría.

### 7. Security Protocols
- **Privacy Mode (Apple Blur):** Interruptor global que aplica un desenfoque de cristal a todos los valores sensibles con un solo clic.
- **Bank-Grade Data:** Infraestructura basada en PostgreSQL con Triggers de integridad y encriptación AES-256 (via Supabase).

## 🛠️ Stack Tecnológico
- **Frontend:** React 19 + Vite.
- **Estilos:** Tailwind CSS v4 + Framer Motion.
- **Estado:** Zustand (Global State Management).
- **Backend:** Supabase (PostgreSQL + Auth + RLS).
- **Branding:** Centralized Brand Engine (Favicon API).

---

## 🚀 Despliegue e Instalación
1. Clonar repositorio: `git clone https://github.com/BLOCKHORN/chillcalc`
2. Instalar dependencias: `npm install`
3. Configurar variables de entorno en `.env`:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
   - `VITE_FINNHUB_API_KEY`
4. Iniciar terminal: `npm run dev`
